## 프로젝트 소개

우리집에서 지금 당장 만들 수 있는 레시피, **home recipe**입니다!

## 핵심 기능

**냉장고 속 수많은 재료들 사이에서 뭘 먹을지 고민은 끝**

나의 냉장고 속 재료들을 토대로 오늘 맛있게 먹을 여러가지 레시피를 추천해드려요!
